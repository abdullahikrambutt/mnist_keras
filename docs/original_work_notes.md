# Original work notes

This project was recreated from an MNIST/Keras learning notebook. The source notebook included:

- a fully connected neural network for MNIST digit classification,
- a convolutional neural network with convolution, pooling, batch normalization, dropout, and augmentation,
- examples for plotting correct/incorrect predictions,
- examples for visualizing convolutional feature maps.

The recreated project avoids copying the full notebook prose and uses a more maintainable GitHub project structure.
